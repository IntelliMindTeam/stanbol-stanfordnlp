The Apache Stanbol Stanford NLP Integration

This Server provides in integration for Stanford NLP with the Apache Stanbol Enhancer NLP module.

Stanford NLP is used for Sentence detection, Tokenizing and Part of Speech, Named Entiy Recognition and Lemmatization of texts. Supported languages do depend on the configuration of the Server. But in principle all languages supported by Stanford NLP can also be supported by this integration.
